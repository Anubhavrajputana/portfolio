My Projects

College ERP System

I developed a comprehensive College ERP System tailored for managing academic and administrative operations efficiently. The system integrates key modules including: Student Management – Automated student registration, course allocation, and academic profiles. Fees Management – Secure online and offline payment options with QR-based transactions, auto-generated receipts, defaulter lists, and custom reports. Faculty & Admin Panels – Role-based dashboards for teachers, accountants, and admins with access control. Notification System – Real-time alerts via WhatsApp and email for fees, exam updates, and attendance. Analytics Dashboard – Interactive graphs, charts, and reports for decision-making insights. Modern UI – Built using React.js, Node.js, MongoDB, and styled with Tailwind CSS using a sleek dark glassmorphism theme.

E-Recycle

Developed a web-based E-Recycle System that enables users to register, list recyclable materials, and track collection activities. The platform supports multiple material categories including glass, copper, bronze, iron, and paper. It features secure user authentication, an intuitive UI with Font Awesome icons and Google Fonts, and a built-in payment gateway with QR code-based online transactions and offline payment support. JavaScript was used for frontend logic and validation, enhancing overall user experience.

E-commerce

Developed a fully functional online grocery delivery platform that allows users to browse, search, and order daily essentials with ease. The system supports user authentication, product categorization, real-time cart updates, and order tracking. Integrated a secure payment gateway for seamless transactions and used MongoDB for dynamic data storage. Designed a responsive UI using React.js and Bootstrap for an optimized user experience across all devices.

Virtual Assistant

I developed an AI-powered Virtual Assistant capable of understanding both voice and text commands. It performs tasks like information retrieval, scheduling reminders, searching the web, and basic conversation. Using NLP libraries like spaCy and speech recognition, the assistant processes user input and responds accordingly using real-time text-to-speech. This project enhanced my understanding of human-computer interaction, natural language understanding, and real-time API integration.
